一、大语言模型（LLM）基础 ​

教学目标：掌握大模型相关基础概念，轻松且深入的理解 机器学习 & GPT 核心原理

  1. 大模型基础概念讲解 & 开发环境搭建
  • ChatGPT 是什么、现状；ChatGPT 与 GPT、LLM 关系；用 GPT 可以做些什么；其他 LLM 产品介绍
  • OpenAI API 介绍、API Key 获取、计费逻辑、Playground；
  • AI、AGI、AIGC、AI Agent 等概念讲解
  • OpenAI GPTs 构建一款 AI App；搭建开发环境，跑通 "Hello World"
  1. 机器学习 & GPT 原理介绍
  • 人工智能发展史、机器学习、神经网络、深度学习、模型训练等基础概念讲解
  • 从 RNN 到 Seq2Seq 再到 Transformer 的算法架构原理与演变过程
  • Tokenizer、Embedding 原理解析
  • GPT、Bert、T5 的技术选型

二、Prompt 提示工程 ​

教学目标：介绍什么是 Prompt 以及 Prompt 的重要性，从 Prompt 入门到进阶，系统性的掌握 Prompt 设计技巧

  1. Prompt 设计核心要素（入门）
  • 介绍 Prompt 以及其重要性
  • 介绍 Prompt 设计的四大要素（GPT 角色设定、任务背景铺垫、任务描述、任务过程示例）
  • 介绍 Prompt 的设计原则
  1. Prompt 的结构化（进阶）
  • 介绍 Prompt 的结构化以及其优势
  • 介绍 Prompt 结构化的流派以及不同流派所其适合的应用场景
  1. Prompt 的思维链（高阶）
  • 介绍思维链，利用思维链突破 GPT 的推理能力上限，处理更加复杂的推理任务
  • 介绍更多的思维链方式，增强 GPT 答复的稳定性，减少 GPT 的幻觉产生概率
  1. Prompt 的多路协同（高阶）
  • 介绍 Prompt 的多路协同，利用多路协同完成需多角色参与的复杂任务（如软件开发）
  1. 实战
  • 设计并构建自动化测试专家（分析代码、找到代码 BUG、返回修改建议）
  • 设计并构建文字小游戏（成语接龙、猜数字、石头剪刀布）

三、基础 ​

教学目标：全面掌握大模型应用开发基础能力，详解 OpenAI API、GPT Tools、Assistants 等核心机制

  1. OpenAI API 详解
  • 详解 OpenAI API，掌握核心接口：ChatCompletion、Embeddings、Audio、Images 等
  • 利用 SDK 完成相关接口精细化调用
  1. GPT Tools 机制
  • 详解 GPT Tools 机制，API 定义、与 GPT 交互流程
  • 利用 GPT Tools 开发 GPT 联网插件，为 GPT 赋予联网能力
  1. Assistants 详解
  • 详细解析 Assistants 工作原理
  • 基于 Assistants 构建电商场景的客服/售后维修客服

  1. 知识储备
  • Gradio & FastAPI 快速入门

  • 系统特色分析 & 整体架构设计
  • 详细流程设计
  1. ChatBot 对话能力构建

五、进阶 ​

教学目标：掌握大模型应用开发的进阶能力，系统性学习 RAG 架构、大模型应用开发框架、大模型多模态

  • RAG 效果评估体系构建
